# Building with Claude

> You've got specs and plans in hand — now it's time to execute. Today is hands-on: by the
> end you'll have turned your own plan into real, working output. We'll cover how to turn a
> plan into an ordered roadmap, figure out what can run in parallel versus what has to wait,
> then set Claude loose on large chunks of the work autonomously — with the right checkpoints
> so you stay in control of the result.

This is a **five-hour, in-person** day. It teaches the *execution* half of the
[lifecycle](/docs/working-with-claude): you arrive with a plan and leave having run it.

## Before you come

This day assumes a working setup and a plan in hand. **Complete the
[pre-flight checklist](./preflight.md) the day before** — every box must pass, or the first
hour gets eaten by environment troubleshooting instead of building.

A terminal is required for the execution exercises. The Claude desktop app is fine for the
conversational ones (roadmap, sequencing), but worktrees and autonomous runs are a terminal
workflow.

## Run of show

The meal lands mid-afternoon — a late lunch / early dinner — right after the first autonomous
run, so the highest-energy conversation of the day spills into the break instead of getting
cut off.

| Time | Block |
|---|---|
| 12:00 | **Welcome & the day's arc** — where specs/plans leave off, what "done" looks like by 5pm |
| 12:10 | **Plan → roadmap** — read a plan back to Claude as an ordered backlog · *Exercise 1* |
| 12:45 | **Sequencing & dependencies** — what blocks what, what's parallelizable · *Exercise 2* |
| 1:15 | **Break** |
| 1:25 | **Isolation: worktrees** — a safe, throwaway workspace per run · *Exercise 3* |
| 1:50 | **First autonomous execution** — hand Claude one task, watch it run · *Exercise 4* |
| 2:30 | **Debrief Exercise 4** — what went right, what went sideways |
| 2:45 | **Late lunch / early dinner** |
| 3:30 | **Scaling execution** — larger chunks, when to split · *Exercise 5* |
| 4:10 | **Checkpoints & verification** — review gates, catching "done" that isn't |
| 4:20 | **Capstone co-working** — execute real plan material; facilitators float |
| 4:50 | **Close** — what each person shipped, what to run solo next |
| 5:00 | End |

## The exercises

The five exercises are the spine of the day — each one runs against *your own plan*, so you
leave with real progress. **[Full details: the exercises](./exercises.md)**.
