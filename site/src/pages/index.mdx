# Learn spec-driven development with an agentic workspace

This handbook is a hands-on skills-building exercise. You set up an agentic
workspace — Claude Code, persistent context files, and rules that load into
every session — then practice the spec-driven workflow on a sandbox project:
spec, plan, subagent build, review.

## What you end up with

- A **working agentic workspace**: Claude Code wired to a personal `CLAUDE.md`,
  a workspace map, and a linked rule set that keeps every session consistent
- The **spec-driven development workflow** — spec → plan → subagent build →
  review — practiced end to end on a low-stakes sandbox project
- A **mission-control surface you built yourself** — the small dashboard-style
  app that is the exercise's learning vehicle: every skill above gets practiced
  by building it, in your own disposable repo
