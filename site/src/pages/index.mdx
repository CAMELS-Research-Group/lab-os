# Learn spec-driven development with an agentic workspace

This handbook is a hands-on skills-building exercise. You set up an agentic
workspace — Claude Code (Anthropic's AI coding assistant), persistent context
files, and rules that load into every session — then practice spec-driven
development (writing down what to build, and the plan to build it, before any
code) on a sandbox project: spec, plan, subagent build (implementation
delegated to helper AI agents), review.
